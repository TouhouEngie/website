About the Creator

Hi. I bike, code, draw, and drum in the slivers of spare time that I have.

I'm an aspiring computer engineering student, teaching myself about computers and coding since August 2020.

Most of my expertise stems from HTML, CSS, Tailwind CSS, and JavaScript, with a little bit of Godot.

Currently learning React, C, and PHP/mySQL to build this more...

Designing front-end using unorthodox software, with the skills primarily for CLI programs and a slight obsession with vocaloid and PJSK

I also run my school's one and only coding club, powered by and partnered with Hack Club. (shoutout to their great programs btw)
